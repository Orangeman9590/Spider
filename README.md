# Spider (Works in Linux, Windows and Mac)
HOW TO USE (Linux)
-----------
1. Clone Repo
```
$ git clone https://github.com/Orangeman9590/Spider
```
--------------
2. Change Directory
```
$ cd Spider
```
---------------
3. Install Requirements
```
$ pip install -r requirements.txt
```
----------------
4. Run
```
$ python3 spider.py
```
